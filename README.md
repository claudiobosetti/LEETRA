# LEETRA: 1337 Python Translator :snake: :nerd_face:

A Python tool to translate any language to 1337  :snake: :fire:

## Requirements :page_facing_up: 

* Python 3.7+
* termcolor

## Installation :gear:

```bash
py -m pip install termcolor
python3 -m pip install termcolor
```
## Run :runner:

```bash
py main.py

python3 main.py
```
